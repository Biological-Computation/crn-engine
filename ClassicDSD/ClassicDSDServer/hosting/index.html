Install instructions

  1. Install .net for your platform
    • Windows:https://www.microsoft.com/net/download/thank-you/net472
    • macOS or Linux:http://www.mono-project.com/download/
  2. Unzip the release
  3. Run on terminal
      Windows: ClassicDSDServer.exe
      On macOS or Linux: mono ClassicDSDServer.exe
